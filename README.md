## obsah
- **bms.csv** obsahuje agregovaná vyčištěná data z logu BMS LFP baterie - vstupní soubor pro predikční skript
- **meteo.csv** vstupní data z openmeteo.com očištěná pomocí profilu horizontu z LIDAR transformace, předpověď relevantních dat 7 dní zpět, 7 dní dopředu pro fotovoltaický systém a baterie LFP
- **forecast.csv** - finální výstupní soubor s predikcí úrovně nabití baterie
